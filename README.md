# eva2
🏔 Aysén Vivo

Plataforma de Experiencias Turísticas Sostenibles de la Región de Aysén, Patagonia.

Proyecto desarrollado para la Evaluación 2 — UA2: Interacción con documentos HTML usando JavaScript, asignatura Programación Front End (TI3V31), INACAP.

Descripción

Aysén Vivo conecta a viajeros con emprendedores locales de Coyhaique, Puerto Aysén, Chile Chico, Caleta Tortel y Cochrane, difundiendo experiencias de turismo de bajo impacto (navegación, trekking, pesca y cultura) y controlando el cupo máximo de cada una.
